# ユーザーガイド

> Todica を使う人向けのドキュメント入口。

## Todica とは

"今日やること" だけにフォーカスした、完全に個人用のタスク管理アプリです。プロジェクトという文脈を保ちつつ、最小手数でタスクを足し、今日のタスクから 1 つを選んで集中して消化していく一本道のワークフローを提供します。

詳しい思想は [プロジェクト概要](../developer/project.md) を参照してください。

## 対象読者

- Todica を自分の道具として使いたい個人
- 自分でサーバを立てて使う（または Android のローカルモードで使う）ことを想定

## 基本ワークフロー

1. プロジェクトを作る（任意。単発タスクのみでも使える）。
2. タスクを足す（期限は既定で今日。今日 / 明日のみ選択可）。
3. 今日のタスクから 1 つを「現在のタスク」に選ぶ → それだけが大きく表示される。
4. 完了すると自動で次のタスクが現在のタスクになる。
5. 日付が変わると未完了タスクは翌日へ繰越され、ゴミ箱は自動で空になる。

繰り返しタスクは「ルーティン」として登録すると、指定曜日に自動で今日のタスクに入ります。
