# ポリシー適合確認チェックリスト

Google Play Store 公開前に確認すべきポリシー適合項目のチェックリスト。

## アプリ技術要件

- [ ] targetSdkVersion が Google Play の要件（最新メジャーリリース時点の基準）を満たしている
- [ ] ターゲット API レベルが審査基準を満たしている
- [ ] パーミッション宣言が AndroidManifest.xml に正しく記述されている
- [ ] 不要なパーミッションが含まれていない

## コンテンツレーティング

- [ ] IARC コンテンツレーティングアンケートに回答済みである
- [ ] IARC レーティングが適切なカテゴリ（全年齢向け）に設定されている

## プライバシー・データ関連

- [ ] プライバシーポリシーが公開 URL で参照可能な状態になっている
- [ ] プライバシーポリシーの URL が Google Play Console に正しく入力されている
- [ ] プライバシーポリシーの内容がアプリの実際のデータ取り扱いと一致している
- [ ] データセーフティセクションの申告内容が実態と一致している
- [ ] データセーフティフォームへの入力が完了している

## ストア掲載情報

- [ ] アプリ名・短い説明・詳細説明が入力されている
- [ ] スクリーンショットがアップロードされている
- [ ] フィーチャーグラフィック（1024 × 500）がアップロードされている
- [ ] アイコン（512 × 512）がアップロードされている
- [ ] カテゴリが設定されている
- [ ] 連絡先メールアドレスが設定されている

## 不正行為ポリシー

- [ ] 不正行為ポリシーに違反するコンテンツが含まれていない
- [ ] 過大なパーミッション要求がない（AndroidManifest.xml で宣言するパーミッションは必要最小限）
- [ ] スパム・繰り返しコンテンツの掲載がない

## 広告ポリシー

- [ ] 広告 SDK を使用していない（本アプリは広告なし）
- [ ] 子供向け広告ポリシーの対象外であることを確認（全年齢向けだが広告なし）

## リリース前最終確認

- [ ] テストトラック（内部テスト）での動作確認が完了している
- [ ] クラッシュやエラーがないことを確認している
