# データセーフティ

Google Play Store のデータセーフティセクションに申告する情報をまとめる。

## 収集するデータ

本アプリが外部サーバへ収集するデータはありません。

ユーザーが入力するタスクデータはすべてユーザー自身の管理下（端末内またはユーザー運用サーバ）に保存されます。開発者はこれらのデータにアクセスできません。

## 共有するデータ

第三者と共有するデータはありません。広告ネットワーク・分析サービス・外部 API へのデータ送信は一切行いません。

## 暗号化

データの暗号化は Android システム標準の暗号化機能に従います。端末の暗号化設定が有効な場合、保存されたデータも暗号化されます。

## 削除

ユーザーはいつでもデータを削除できます。

- アプリ内から個別のタスクを削除できます。
- アプリをアンインストールすることで、端末内のすべてのデータが削除されます。
- サーバモードをご利用の場合は、ユーザー自身がサーバ側のデータを削除してください。

削除リクエストへの対応: ユーザー自身がアプリ内またはアンインストールにより削除を完結できるため、開発者への削除申請は不要です。

## 独立監査

データセーフティの内容について独立監査は実施していません。本アプリは外部サーバへのデータ収集を行わないため、第三者監査の対象となるデータ処理が存在しません。
