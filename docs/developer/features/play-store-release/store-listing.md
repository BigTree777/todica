## 詳細説明

Todica は「今日やること」に集中するためのシンプルなタスク管理アプリです。

### 主な機能

- 今日ビュー: その日にやるべきタスクだけを表示し、集中力を高めます。
- タスクの作成・編集・完了管理が直感的な操作で行えます。
- ローカルモードとサーバモードを選択できます。
  - ローカルモード: インターネット接続不要、端末内にデータを保存します。
  - サーバモード: 自分でサーバを立てて複数端末で同期できます。
- データは完全にユーザー管理下に置かれます。外部への送信は一切ありません。

### プライバシー

個人データを外部サーバへ収集しません。詳細はプライバシーポリシーをご確認ください。

## スクリーンショット要件

### キャプチャすべき画面一覧

1. 今日ビュー（メイン画面）
2. タスク作成画面
3. タスク詳細・編集画面
4. 設定画面

### アセット仕様

| アセット | サイズ | 形式 |
|---------|--------|------|
| フィーチャーグラフィック | 1024 × 500 px | PNG / JPG |
| アイコン | 512 × 512 px | PNG（32 bit、アルファチャンネルあり） |
| スクリーンショット（縦） | 最小 320 px, 最大 3840 px | PNG / JPG |

フィーチャーグラフィックは 1024 px 幅 × 500 px 高さで作成すること。
アイコンは 512 px × 512 px（32 bit PNG、アルファチャンネルあり）で作成すること。
スクリーンショットは最低 2 枚・最大 8 枚を用意すること。
